Imagens opcionais da pagina de capitulos
========================================

Coloque os arquivos com estes nomes:

- `assets/img/book-cover.jpg`: capa do livro.
- `assets/img/chapters/banner-capitulos-d-o.png`: imagem larga de fundo do topo da pagina de capitulos.
- `assets/img/chapters/cap-01.jpg`: imagem do capitulo 1.
- `assets/img/chapters/cap-02.jpg`: imagem do capitulo 2.
- Continue o mesmo padrao ate `cap-30.jpg`.
- `assets/img/chapters/thumbs-capitulos-d-o.png`: prancha original usada para recortar as thumbs.
- `assets/img/chapters/banner-videos-d-o.png`: imagem larga de fundo do topo da pagina de videos.
- `assets/img/chapters/thumbs-videos-d-o.png`: prancha original usada para recortar as thumbs dos videos.
- `assets/img/videos/video-01.jpg`: poster do video 1.
- Continue o mesmo padrao ate `video-06.jpg`.

Formatos recomendados:

- Capa: JPG vertical, proporcao aproximada 2:3.
- Hero: JPG horizontal, pelo menos 1600x700.
- Capitulos: JPG horizontal, proporcao aproximada 16:9.

Se algum arquivo nao existir, o app usa o visual gerado por CSS como fallback.
