# educa-one — PWA

Estrutura inicial pronta para receber os conteúdos.

## Como testar

Abra a pasta com um servidor local. No VS Code, use a extensão Live Server. Outra opção:

```bash
python -m http.server 8080
```

Depois acesse `http://localhost:8080`.

## Conteúdo

Edite `data/site.json`. O projeto já contém 20 capítulos, 4 vídeos, 5 perguntas e 5 flashcards por capítulo.
